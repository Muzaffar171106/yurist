# Xavfsizlik

- Production server reverse proxy va HTTPS ortida ishlashi kerak.
- `.env` repozitoriyga qo'shilmaydi.
- Savol serverda saqlanmaydi; loglarda savol matni yozilmaydi.
- API so'rovlari tezligi cheklangan.
- CSP, Helmet, request ID va cheklangan JSON hajmi yoqilgan.
- `data/feedback.jsonl` fayliga kirish faqat server administratorida bo'lishi kerak.

Zaiflik topilsa, public issue o'rniga loyiha egasining maxsus xavfsizlik elektron pochtasiga yuborish tartibini kiriting.
